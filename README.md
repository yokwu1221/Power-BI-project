Data Professional Career Survey Analysis

Project Overview:
This project features a comprehensive Power BI dashboard analyzing a global survey of data professionals (Data Analysts, Data Scientists, Engineers, etc.). The analysis focuses on uncovering key trends in salary structures, programming language preferences, job satisfaction levels, and the perceived barriers to entry within the data industry.

Technical Highlights:
ETL & Data Transformation: Leveraged Power Query to perform extensive data cleaning. This involved handling missing values, standardizing inconsistent job titles, and normalizing country classifications to ensure data integrity.

Data Architecture: Implemented a Star Schema to optimize filter performance and ensure a scalable data model.

Interactive Visualization: Designed a user-centric interface incorporating dynamic switchers, drill-through actions, and tooltips to provide multi-dimensional insights at a glance.
